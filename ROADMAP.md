Linua Updater — Roadmap

this document outlines planned improvements and future development goals for the project.

upcoming features (planned)
1. integrity check (sha256) for all dlc

verify downloads before extraction

detect corrupted or altered files

reduce user-side issues

2. improved logging system

color-coded logs

grouped logs per dlc

optional log.txt export

clearer error messages

3. retry system for unstable connections

automatic re-tries on failed downloads

better handling of timeouts and resets

4. pre-installation diagnostics

disk space check

7-zip availability test

permission check

folder structure validation

5. batch install profiles

install all EPs

install all GPs

install packs by category

minimal recommended set

6. history tab

local record of previous actions

successful installs, cancelled operations, repair history

7. expanded repair system

validate installer folders

detect corrupted dlc data

fix missing directories

8. connection test

small test-file download

classify user connection: good / unstable / blocked

help diagnose VPN issues

long-term ideas

(these may be added later depending on priorities)

9. full UI overhaul

sidebar navigation

separate sections: install / repair / settings / logs

modern linua-style theme

10. settings menu

toggle update checks

choose language

safe mode

custom download directory

11. built-in help

mini-documentation inside the app

“troubleshooting” and EP03/EP06 guides

12. optional mirror fallback system

automatic switch to secondary download servers if GitHub is slow/unavailable

requires hosting support (future consideration)

development philosophy

stable over experimental

safety first

clear logs, clear errors

no modifications to game binaries

no interaction with EA servers

feedback

feature suggestions are welcome via GitHub Issues.
please do not request illegal functionality (online bypass, unlockers, multiplayer patches).
